# Pro Gamer Physical Test

C언어 콘솔 기반 미니게임 프로젝트 기획 자료입니다. 반응속도, 방향키 순발력, 클릭, 타이밍 테스트를 진행하고 최종 점수에 따라 플레이어의 티어를 판정하는 게임을 목표로 합니다.

## 폴더 구조

```text
.
├── docs/
│   ├── Pro_Gamer_Physical_Test_게임기획서.docx
│   └── rendered/
│       └── Pro_Gamer_Physical_Test_게임기획서.pdf
├── presentation/
│   ├── Pro_Gamer_Physical_Test_중간발표.pptx
│   └── Pro_Gamer_Physical_Test_발표대본.md
└── scripts/
    ├── make_game_plan_docx.py
    └── render_pdf_pages.swift
```

## 발표 자료

- `presentation/Pro_Gamer_Physical_Test_중간발표.pptx`: 중간발표용 PPT
- `presentation/Pro_Gamer_Physical_Test_발표대본.docx`: Word에서 열기 쉬운 발표 대본
- `presentation/Pro_Gamer_Physical_Test_발표대본.txt`: 메모장/텍스트 편집기용 발표 대본
- `presentation/Pro_Gamer_Physical_Test_발표대본.md`: Markdown 형식 발표 대본

## 기획서

- `docs/Pro_Gamer_Physical_Test_게임기획서.docx`: 원본 게임 기획서
- `docs/rendered/Pro_Gamer_Physical_Test_게임기획서.pdf`: 렌더링 확인용 PDF

## 참고

`scripts/` 폴더의 파일은 기획서 생성과 PDF 페이지 렌더링에 사용한 보조 스크립트입니다.
